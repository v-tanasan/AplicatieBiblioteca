# AplicatieBiblioteca
Aplicatie in C# pentru o biblioteca

Program pentru gestionarea împrumuturilor de cărți dintr-o bibliotecă. O carte poate să existe în mai mult exemplare. Pe lângă operațiile de adăugare, editare, ștergere pentru un autor sau carte sau pentru persoana care împrumută, se vor afișa următoarele situații:
- pentru o anumită carte se va afișa dacă este disponibilă sau nu;
- pentru o anumită persoană se va afișa numărul de cărți împrumutate; dacă numărul depășește o anumită valoare stabilită în secțiunea de configurări, va apare un mesaj de atenționare și nu se va permite împrumutul unei alte cărți.

Diagrama claselor ce vor fi implementate este prezentata in fisierul Diagrama-clase.pdf (generat cu https://www.planttext.com/).
